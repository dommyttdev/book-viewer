# Manga

自炊本閲覧Webアプリケーションの開発リポジトリです。

プロダクト、アーキテクチャ、設計、開発、テスト、運用の情報は `doc/` 配下で管理します。
重複管理を避けるため、このREADMEはリポジトリの入口とし、仕様や技術判断の詳細は各ドキュメントを正とします。

```text

```
