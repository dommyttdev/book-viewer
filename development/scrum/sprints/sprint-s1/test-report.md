## 実行予定

| 種別 | コマンドまたは確認内容 | 目的 |
| --- | --- | --- |
| API単体テスト | `.\gradlew.bat :apps:api:test` | 認証ドメイン、トークン、チャレンジ、セッション状態のテストを実行する。 |
| API結合テスト | 実装時に追加するDB永続化テスト | PostgreSQLを正本とした登録、確認、ログイン、セッション失効を確認する。 |
| フロントエンド確認 | `npm.cmd run lint`, `npm.cmd run typecheck`, `npm.cmd run build` | 認証画面または確認導線の静的確認を行う。 |
| 手動確認 | local profileで登録からログアウトまで確認 | 開発環境でメール内容、Cookie、ログ安全性を確認する。 |
