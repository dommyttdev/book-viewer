# リリースノートテンプレート

## 目的

このテンプレートは、自炊本閲覧Webアプリケーションのリリース内容、既知の問題、移行作業を記録するために使用する。

## 記録ルール

- 利用者や運用者に影響する変更を中心に記録する。
- 内部実装だけの変更でも、運用、データ、セキュリティ、性能に影響する場合は記録する。
- 秘密情報、内部物理パス、不要な個人情報は記録しない。
- 既知の問題は、影響範囲と回避策を明記する。
