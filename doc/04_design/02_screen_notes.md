# 画面メモ

## 目的

このドキュメントは、自炊本閲覧Webアプリケーションの主要画面について、初版の役割、表示項目、主要操作、状態、注意点を整理する。

詳細なワイヤーフレームやコンポーネント設計は、実装前または実装中に更新する。

## 共通方針

- 画面は一般ユーザ向け領域と管理ユーザ向け領域を分ける。
- 物理ファイルパス、`storage_key`、トークン、セッションID、内部エラー詳細は表示しない。
- フロントエンドの入力チェックはUXのために行い、正本の検証はバックエンドで行う。
- 一覧、検索結果、管理一覧はページングまたは追加読み込みを前提にする。
- 大量画像や大きな一覧を一度に読み込まない。
- 権限がない操作は表示しない。ただし直接アクセス時はサーバ側の権限判定結果を安全に表示する。
- エラー表示は利用者に必要な行動が分かる範囲に留め、秘密情報や内部詳細を出さない。

## ログイン画面

### 役割

一般ユーザがログインし、本一覧画面へ入るための画面。

### 表示項目

| 項目 | 内容 |
| --- | --- |
| メールアドレス | 初期MVPの識別子優先フローで使用する。 |
| Passkey認証 | ブラウザ / OSのPasskey認証へ進む。 |
| 会員登録導線 | 未登録ユーザを会員登録画面へ誘導する。 |
| アカウント復旧導線 | Passkeyを使えない場合に使用する。 |

### 主要操作

- ログインする。
- Passkey認証を行う。
- 会員登録画面へ移動する。
- アカウント復旧へ移動する。

### 状態とエラー

| 状態 | 表示方針 |
| --- | --- |
| 初期表示 | メールアドレス入力とPasskey認証開始導線を表示する。 |
| Passkey認証待ち | ブラウザ / OSの認証結果待ち状態を表示する。 |
| 認証失敗 | アカウント存在推測を避ける安全な文言にする。 |
| メール未認証 | メール認証再送導線を表示する。 |
| レート制限 | 再試行まで待つ案内を表示する。 |

## 会員登録画面

### 役割

一般ユーザがメール認証付きで会員登録するための画面。

### 表示項目

| 項目 | 内容 |
| --- | --- |
| メールアドレス | 登録とメール認証に使用する。 |
| 表示名 | 本人表示用の名前。 |
| メール認証待ち表示 | 登録後に確認メール送信済みであることを表示する。 |
| Passkey登録導線 | メール認証後にPasskey登録へ進める。 |

### 主要操作

- 会員登録する。
- メール認証を完了する。
- Passkeyを登録する。
- 認証メールを再送する。
- ログイン画面へ戻る。

### 注意点

- 登録済みメールかどうかを推測しやすい表示にしない。
- トークン値を画面上に表示しない。

## 本一覧画面

### 役割

一般ユーザが閲覧可能な本を探し、詳細またはビューアへ移動するための画面。

### 表示項目

| 項目 | 内容 |
| --- | --- |
| 表紙サムネイル | `thumbnailUrl`から取得する。未生成時は代替表示。 |
| タイトル | 本詳細への主導線。 |
| 著者 | 複数著者を表示できる形式にする。 |
| タグ | 主要タグを表示する。 |
| シリーズ | シリーズ名と順序を表示する。 |
| ページ数 | 閲覧ボリュームの目安。 |
| 読みかけ状態 | 最後に読んだページがあれば表示する。 |
| お気に入り状態 | 登録済みか分かる表示にする。 |

### 主要操作

- 本詳細を開く。
- 続きから読む。
- 検索する。
- お気に入り登録または解除する。
- ページングまたは追加読み込みを行う。

### 状態とエラー

| 状態 | 表示方針 |
| --- | --- |
| 読み込み中 | 一覧領域に読み込み状態を表示する。 |
| 一覧なし | 閲覧可能な本がない空状態を表示する。 |
| サムネイル取得失敗 | 代替表示に切り替える。 |
| 認証切れ | ログイン画面へ誘導する。 |

## 本詳細画面

### 役割

本の詳細情報を確認し、読む操作とお気に入り操作を行う画面。

### 表示項目

| 項目 | 内容 |
| --- | --- |
| 表紙 | サムネイルまたは代表画像を表示する。 |
| タイトル | 本の主表示名。 |
| 説明 | 管理ユーザが登録した説明。 |
| 著者 | 著者一覧。 |
| タグ | タグ一覧。 |
| シリーズ | シリーズ名と巻数または順序。 |
| 種別 | コミック、小説など。 |
| ページ数 | 変換済みページ数。 |
| 閲覧履歴 | 最後に読んだページと最終閲覧日時。 |

### 主要操作

- 1ページ目から読む。
- 続きから読む。
- お気に入り登録または解除する。
- 一覧または検索結果へ戻る。

### 注意点

- 閲覧不可、非表示、論理削除済みの本は一般ユーザに表示しない。
- 変換未完了の本は一般ユーザの閲覧対象外にする。

## 本アップロード画面

### 役割

管理ユーザが自炊本アーカイブをアップロードし、変換ジョブを作成する画面。

### 表示項目

| 項目 | 内容 |
| --- | --- |
| 原本アーカイブ | zip / rar / 7zipを受け付ける。 |
| タイトル | 必須項目。 |
| 説明 | 任意項目。 |
| 著者 | 既存著者選択を候補とする。 |
| タグ | 既存タグ選択を候補とする。 |
| シリーズ | 既存シリーズ選択を候補とする。 |
| シリーズ内順序 | 任意。 |
| 種別 | 種別コードを選択する。 |
| 変換ジョブ状態 | アップロード成功後に表示する。 |

### 主要操作

- 原本アーカイブを選択する。
- メタ情報を入力する。
- アップロードする。
- 変換状態確認へ移動する。

### 状態とエラー

| 状態 | 表示方針 |
| --- | --- |
| アップロード中 | 二重送信を防止する。 |
| 非対応形式 | ファイル形式エラーを表示する。 |
| 入力不正 | 該当項目へエラーを表示する。 |
| ジョブ作成成功 | ジョブIDと状態を表示する。 |
| ジョブ作成失敗 | 再試行できる導線を表示する。 |

## メタ情報編集画面

### 役割

管理ユーザが本の検索、一覧、詳細表示に使うメタ情報を編集する画面。

### 表示項目

| 項目 | 内容 |
| --- | --- |
| タイトル | 必須。 |
| 説明 | 任意。 |
| 著者 | 複数選択を候補とする。 |
| タグ | 複数選択を候補とする。 |
| シリーズ | 任意。 |
| シリーズ内順序 | 任意。 |
| 種別 | 任意。 |
| 公開状態 | `draft`, `available`, `hidden`など。 |
| 検索インデックス状態 | 後続詳細化候補。 |

### 主要操作

- メタ情報を保存する。
- 管理本詳細へ戻る。
- 変換状態を確認する。

### 注意点

- 検索対象項目の変更後はElasticsearch更新が必要になる。
- PostgreSQLを正本とし、検索更新失敗時も再試行可能な状態を表示できるようにする。

## 検索結果画面

### 役割

一般ユーザがタイトル、著者、タグ、シリーズなどで本を探す画面。

### 表示項目

| 項目 | 内容 |
| --- | --- |
| 検索語 | キーワード入力。 |
| 絞り込み条件 | タグ、著者、シリーズ、種別など。 |
| 並び順 | 関連度、更新日時、タイトルなどを候補とする。 |
| 結果一覧 | 本一覧と同等のカードまたは行表示。 |
| 件数 / ページング | 大量結果に対応する。 |

### 主要操作

- 検索条件を入力する。
- 絞り込みを変更する。
- 本詳細へ移動する。
- ビューアへ移動する。

### 状態とエラー

| 状態 | 表示方針 |
| --- | --- |
| 検索前 | 最近更新された本または検索入力中心の表示を候補とする。 |
| 結果なし | 条件変更の導線を表示する。 |
| Elasticsearch障害 | 安全なエラー表示にし、内部詳細は出さない。 |

## ビューア画面

### 役割

一般ユーザが変換済みWebP画像を読み、閲覧位置を保存する画面。

### 表示項目

| 項目 | 内容 |
| --- | --- |
| ページ画像 | `image/webp`を表示する。 |
| ページ番号 | 現在ページと総ページ数。 |
| 前ページ / 次ページ | 主要操作。 |
| 本詳細へ戻る導線 | 読書中断時に戻る。 |
| 表示モード | 1ページ表示を基本とし、見開きは後続候補。 |
| 拡大縮小 | 後続候補。 |

### 主要操作

- 次ページへ進む。
- 前ページへ戻る。
- ページ番号を指定する。
- 本詳細へ戻る。
- 閲覧位置を保存する。

### 状態とエラー

| 状態 | 表示方針 |
| --- | --- |
| 画像読み込み中 | ページ領域に読み込み状態を表示する。 |
| ページ取得失敗 | 再読み込み導線を表示する。 |
| 閲覧位置保存失敗 | 読書を妨げすぎない範囲で通知する。 |
| 最初 / 最後のページ | 前後移動ボタンの無効状態を明確にする。 |

### 操作候補

- PCでは左右キーでページ送りする候補を残す。
- スマートフォンではタップ領域またはスワイプでページ送りする候補を残す。
- MVPでは1ページ表示を基本にし、見開きと拡大縮小は後続で詳細化する。

## お気に入り画面

### 役割

一般ユーザがお気に入り登録した本を再閲覧する画面。

### 表示項目

| 項目 | 内容 |
| --- | --- |
| お気に入り本一覧 | 本一覧と同等の主要情報。 |
| 登録日時 | 並び順の候補。 |
| お気に入り解除 | 一覧から解除できる候補。 |

### 主要操作

- 本詳細へ移動する。
- 続きから読む。
- お気に入り解除する。
- 本一覧へ移動する。

### 状態とエラー

| 状態 | 表示方針 |
| --- | --- |
| お気に入りなし | 本一覧への導線を表示する。 |
| 対象書籍が閲覧不可 | 非表示または閲覧不可として扱う。 |
| 解除失敗 | 状態を戻し、再試行できるようにする。 |

## 管理ログイン画面

### 役割

管理ユーザが管理領域へ入るための画面。

### 表示項目

| 項目 | 内容 |
| --- | --- |
| メールアドレス | 管理ユーザ用アカウント。 |
| Passkey認証 | 管理ユーザ用Passkeyで認証する。 |

### 主要操作

- 管理ログインする。
- 管理用Passkey認証を完了する。
- 管理ログアウトする。

### 注意点

- 一般ユーザログインとは入口とセッションを分ける。
- 一般ユーザが管理ログイン画面から管理領域に入れないようにする。

## 管理ユーザ一覧画面

### 役割

管理ユーザを一覧、検索、絞り込み、編集する画面。

### 表示項目

| 項目 | 内容 |
| --- | --- |
| メールアドレス | 管理ユーザ識別。 |
| 表示名 | 管理画面での表示名。 |
| ステータス | `active`, `suspended`, `deleted`など。 |
| ロール | 付与済みロール。 |
| 最終ログイン日時 | 監査と管理の参考情報。 |
| ページング | 大量管理ユーザに対応する。 |

### 主要操作

- 管理ユーザを検索する。
- ステータスまたはロールで絞り込む。
- 管理ユーザを登録する。
- 管理ユーザ編集画面へ移動する。
- 管理ユーザを停止または削除する。

### 注意点

- 管理ユーザ登録、編集、削除は`super_admin`を基本とする。
- 権限がないユーザには操作導線を表示しない。

## 管理ユーザ編集画面

### 役割

管理ユーザの表示名、ステータス、ロールを編集する画面。

### 表示項目

| 項目 | 内容 |
| --- | --- |
| メールアドレス | 登録時に使用する。編集可否は後続で決める。 |
| 表示名 | 管理画面での表示名。 |
| ステータス | active / suspendedなど。 |
| ロール | super_admin, admin, operator, viewerなど。 |

### 主要操作

- 管理ユーザを登録する。
- 管理ユーザを更新する。
- 管理ユーザを停止または削除する。
- 管理ユーザ一覧へ戻る。

### 状態とエラー

| 状態 | 表示方針 |
| --- | --- |
| 重複メール | 入力エラーとして表示する。 |
| 最後の`super_admin`停止 | 保存を拒否し、理由を表示する。 |
| 権限不足 | 操作できないことを表示する。 |

## ロール設定画面

### 役割

管理ロールと権限の対応を確認し、必要に応じて設定変更する画面。

### 表示項目

| 項目 | 内容 |
| --- | --- |
| ロール一覧 | super_admin, admin, operator, viewerなど。 |
| 権限一覧 | `book.upload`, `book.update`, `admin_user.manage`など。 |
| 権限マトリクス | ロールごとの操作可否。 |

### 主要操作

- ロールごとの権限を確認する。
- 権限マトリクスを参照する。
- ロール設定を更新する。

### 注意点

- 初版では固定ロールを基本とし、設定変更APIはBeta / v1.0候補として扱う。
- 権限マトリクスとロール別操作表示を同期する。

## 後続で詳細化する事項

- 画面ごとのワイヤーフレーム。
- フォーム項目の詳細バリデーション。
- UI文言、空状態、エラー文言。
- 管理ダッシュボードと変換状態確認画面の独立画面化。
- 権限マトリクス変更時の操作表示制御。
- ビューアのPC / スマートフォン別操作詳細。
