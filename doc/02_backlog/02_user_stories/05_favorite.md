# ユーザーストーリー: お気に入り

## 目的

一般ユーザがよく読む本や後で読みたい本をお気に入り登録し、お気に入り一覧から再閲覧できるようにする。

## 背景

お気に入りは一般ユーザ自身の読書体験を補助する機能であり、管理ユーザや他の一般ユーザとは共有しない。同一ユーザ、同一書籍の有効なお気に入りは一件までとし、重複登録を防ぐ。

## 利用者

- 主利用者: 一般ユーザ

## ストーリー

### FV-US-001: 一般ユーザとして本をお気に入り登録したい

一般ユーザとして、本一覧、本詳細、またはビューア周辺から本をお気に入り登録したい。なぜなら、よく読む本や後で読みたい本へすぐ戻りたいから。

### FV-US-002: 一般ユーザとしてお気に入りを解除したい

一般ユーザとして、不要になったお気に入りを解除したい。なぜなら、お気に入り一覧を自分にとって有用な状態に保ちたいから。

### FV-US-003: 一般ユーザとしてお気に入り一覧を見たい

一般ユーザとして、お気に入り登録した本を一覧したい。なぜなら、登録済みの本を探し直さずに再閲覧したいから。

### FV-US-004: 一般ユーザとして重複登録を避けたい

一般ユーザとして、同じ本が重複してお気に入り登録されないようにしてほしい。なぜなら、一覧や状態表示が不整合にならないようにしたいから。

## 対象範囲

- お気に入り登録
- お気に入り解除
- お気に入り一覧表示
- お気に入り状態表示
- 同一ユーザ、同一書籍の重複防止
- 閲覧不可になった本の扱い

## 対象外

- お気に入りの共有
- お気に入りへのメモや分類付け
- 管理ユーザのお気に入り
- 他ユーザのお気に入り参照
