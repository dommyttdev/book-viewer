# ユーザーストーリー: 本一覧

## 目的

一般ユーザが閲覧可能な自炊本を一覧から見つけ、詳細確認や読書開始へ進めるようにする。

## 背景

本は管理ユーザが登録し、変換完了後に一般ユーザの閲覧対象になる。一般ユーザは書籍を保持しないため、本一覧は全ユーザ共通の閲覧可能カタログとして扱い、非表示、変換未完了、論理削除済みの本は通常表示しない。

## 利用者

- 主利用者: 一般ユーザ
- 関係者: 管理ユーザ

## ストーリー

### BL-US-001: 一般ユーザとして閲覧可能な本を一覧したい

一般ユーザとして、閲覧可能な本を一覧で見たい。なぜなら、読みたい本をすばやく探して開きたいから。

### BL-US-002: 一般ユーザとして本の概要を確認したい

一般ユーザとして、一覧上で表紙、タイトル、著者、タグ、シリーズ、ページ数を確認したい。なぜなら、詳細画面へ移動する前に読む本を判断したいから。

### BL-US-003: 一般ユーザとして読みかけ状態から再開したい

一般ユーザとして、最後に読んだページがある本を一覧から続きで開きたい。なぜなら、前回の読書位置を探し直さずに読み始めたいから。

### BL-US-004: 一般ユーザとしてお気に入り状態を確認したい

一般ユーザとして、一覧上でお気に入り登録済みか確認したい。なぜなら、よく読む本を識別し、必要に応じてお気に入りを切り替えたいから。

## 対象範囲

- 閲覧可能な本の一覧表示
- 表紙サムネイル、タイトル、著者、タグ、シリーズ、ページ数の表示
- 読みかけページと最終閲覧日時の表示
- お気に入り状態の表示
- 本詳細またはビューアへの遷移
- ページングまたは追加読み込み

## 対象外

- 管理ユーザ向けの本管理一覧
- Elasticsearchを使う詳細検索条件
- 非表示、変換未完了、論理削除済みの本の一般表示
- 一般ユーザによる本アップロード
